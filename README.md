# The PyeLab Intensive Landing Page

A landing page for The PyeLab Intensive - "Plan Your Exit Like a Bo$$" - a 5-day strategic framework for high-achieving women planning their exit from corporate.

## Overview

This is a standalone landing page designed to promote and sell The PyeLab Intensive. The page features a modern, elegant design with smooth animations, embedded CSS/JS, and a responsive layout.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **GSAP Animations**: Scroll-triggered animations using GSAP
- **Embedded Styles**: All CSS and JavaScript are embedded in the HTML file
- **Stripe Integration**: Direct links to Stripe payment pages for different tiers

## Getting Started

1. Clone this repository
2. Open `pyelab-landing.html` in a web browser
3. No build process required - it's a static HTML page

## Dependencies

- **GSAP 3.12.2**: For scroll animations (loaded via CDN)
- **Google Fonts**: 
  - Playfair Display
  - Montserrat

## Pricing Tiers
